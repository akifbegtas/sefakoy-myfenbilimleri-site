# Sefaköy My Fen Bilimleri site başlangıcı

Bu klasör, dershane sitesi için statik ve bağımsız çalışan ilk sürümü içerir.

## Dosyalar

- `index.html`: Sayfa yapısı, SEO bilgileri, içerikler ve iletişim alanları.
- `styles.css`: Responsive tasarım, kartlar, hero, form ve mobil menü stilleri.
- `script.js`: WhatsApp yönlendirmesi, sosyal bildirimler, başarı vitrini, SSS ve asistan davranışları.
